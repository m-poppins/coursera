# coursera
<li> Задача классификации на наборе данных Fashion-MNIST
  Fashion-MNIST – это датасет состоящий из 70000 черно-белых изображений одежды 28х28 пикселей каждое. 60000 из них содержатся в тренировочной выборке, и 10000 – в тестовой.
  В Fashion-MNIST содержится 10 классов разной одежды по аналогии с 10 цифрами из MNIST.
  Была решена задача классификации. Обучены такие модели как : логистическая регрессия, полносвязная нейронная сеть, сверточная нейронная сеть.
  Наилучшее качество, которое было получено: **92,20%**
